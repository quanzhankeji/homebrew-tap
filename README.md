# OmniDock Homebrew Tap

Official Homebrew tap for [OmniDock](https://github.com/quanzhankeji/OmniDock).

## Install

```sh
brew tap quanzhankeji/tap
brew install --cask omnidock
```

If Homebrew asks you to trust non-official taps, run:

```sh
brew trust --tap quanzhankeji/tap
```

OmniDock requires macOS 12.3 or later.

## Upgrade

```sh
brew update
brew upgrade --cask omnidock
```

## Uninstall

```sh
brew uninstall --cask omnidock
```

To remove OmniDock preferences as well:

```sh
brew uninstall --cask --zap omnidock
```

See [Support](https://github.com/quanzhankeji/OmniDock/blob/main/SUPPORT.md)
