# Quanzhankeji Tap

## How do I install these formulae?

`brew install quanzhankeji/tap/<formula>`

Or `brew tap quanzhankeji/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "quanzhankeji/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
